

## 前言

需求：
+ 在小程序里显示地图，并标记地图上的一个坐标点。
+ 管理员可以在后台设置坐标点

![](img/mk-a-01.jpg)

## 小程序显示地图和坐标点

### .wxml文件

```
<view style="display:flex; justify-content:center;" >
  <map id="myMap" style="width: 90%; height: 600rpx;"
  longitude="{{longitude}}"
  latitude="{{latitude}}"
  scale="15" markers="{{markers}}" ></map>
</view>
```
解释：

- longitude ： 经度
- latitude ： 纬度
- scale ： 缩放级别，取值范围为5-18
- markers ： 标记点

### .js 文件

```
data: {
    longitude: 116.404008,
    latitude: 39.914209,
    markers:[]
},

/**
* 生命周期函数--监听页面初次渲染完成
*/
onReady: function () {
    this.mapCtx = wx.createMapContext('myMap')
},

```

从后台接口获取到坐标数据后，要把数据渲染到地图上。

接口主要有两个数据，一个是经度longitude，一个是维度latitude

```
var marker = {
  id: 1,
  latitude: data.latitude,
  longitude: data.longitude
}
var markers = new Array();
markers.push(marker);

this.setData({
  markers: markers,
  latitude: data.latitude,
  longitude: data.longitude
});
```

这样就能得到开头的那种显示效果了。

## 后台设置坐标点

使用百度地图在地图上进行选点，效果如下：

![](img/mk-a-02.png)

首先去百度地图开放平台 http://lbsyun.baidu.com ，创建一个应用，最主要的是申请一个秘钥 —— AK

下面是html的demo代码

```

            var ac = new BMap.Autocomplete( //建立一个自动完成的对象

                    "input": "suggestId",
                    "location": map

            ac.addEventListener("onhighlight", function(e) { //鼠标放在下拉列表上的事件
                var str = "";
                var _value = e.fromitem.value;
                var value = "";
                if (e.fromitem.index > -1) {
                    value = _value.province + _value.city + _value.district + _value.street + _value.business;
                }
                str = "FromItem<br />index = " + e.fromitem.index + "<br />value = " + value;

                value = "";
                if (e.toitem.index > -1) {
                    _value = e.toitem.value;
                    value = _value.province + _value.city + _value.district + _value.street + _value.business;
                }
                str += "<br />ToItem<br />index = " + e.toitem.index + "<br />value = " + value;

            var myValue;
            ac.addEventListener("onconfirm", function(e) { //鼠标点击下拉列表后的事件
                var _value = e.item.value;
                myValue = _value.province + _value.city + _value.district + _value.street + _value.business;
                G("searchResultPanel").innerHTML = "onconfirm<br />index = " + e.item.index + "<br />myValue = " + myValue;

                setPlace();

            function setPlace() {
                map.clearOverlays(); //清除地图上所有覆盖物

                    map.centerAndZoom(pp, 18);
                    map.addOverlay(new BMap.Marker(pp)); //添加标注
                }

                var local = new BMap.LocalSearch(map, { //智能搜索
                    onSearchComplete: myFun
                });
                local.search(myValue);
            }
        </script>
```

### 添加搜索的功能

可以模糊搜索地名快速定位到想要的地点，效果如下：

![](img/mk-a-03.png)

主要使用了百度地图的api，主要是创建一个自动完成的对象，具体代码可以参照上面的demo

```
var ac = new BMap.Autocomplete(
{
    "input": "suggestId",
    "location": map
});
```

最后在两个input元素，lng和lat，就能得到坐标点的经度和维度，保存到数据库就可以提供给小程序调用了。

---

**在使用中发现百度地图得到的坐标数据放到小程序里就会出现不小的偏差，因为小程序里使用的是腾讯地图，（当然也可以选择高德地图），偏差大概有500米，这个是无法忍受的，所以决定后台换成腾讯地图。**
