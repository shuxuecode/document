


## jvm命令

使用java命令

在命令行中，你可以通过以下命令打印出所有的JVM参数和系统属性：
```
java -XX:+PrintFlagsFinal -version
```

这条命令会列出所有的JVM参数（包括默认值）和它们的当前值。-version部分会打印Java版本信息，同时也触发输出JVM参数。
