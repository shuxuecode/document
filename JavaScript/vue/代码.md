
## element ui输入框监听enter回车事件

```
<el-input v-model="keywords" placeholder="多个关键字请以 | 分隔" @keyup.enter.native="onSubmit"></el-input>
```

```
<el-button type="primary" @keyup.enter.native="onSubmit">登陆</el-button>
```
